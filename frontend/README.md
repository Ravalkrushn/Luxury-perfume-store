# Luxury Perfume Store

A simple web application for managing and exploring luxury perfumes.

---

## Features

- Responsive Home page
- Admin Login & Admin Panel
- Add, Edit, Delete perfumes (CRUD operations)
- Image upload support
- Explore Collection page with product cards
- Learn More & About pages
- Mobile-friendly UI

---

## Tech Stack

### Frontend

- React.js
- CSS

### Backend

- Node.js
- Express.js
- MongoDB
